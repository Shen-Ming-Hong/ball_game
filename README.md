# 🎯 槌球遊戲機 (Ball Game Machine)

一個基於 Arduino UNO 的槌球遊戲機系統，整合 5 組 TCRT5000 紅外線感測器、TM1637 四位數顯示器與 MP3 音效模組，採用中斷驅動架構確保精確計時。

## 📋 遊戲機制

### 🎮 玩法說明

1. **等待進球**：系統啟動後顯示燈光繞圈特效，等待玩家投球
2. **首球入洞**：任一感測器偵測到進球，自動啟動 **60 秒計時視窗**，分數開始累加
3. **連續得分**：每次進球，分數依進洞位置累加，計時視窗**重置為 60 秒**
4. **超時結束**：60 秒內無任何進球，遊戲結束，播放結束音效，顯示最終分數 2 秒後**分數歸零**重新等待

### 🏆 得分系統

| 感測器 | 類比引腳 | 數位引腳 | 得分  |
| ------ | -------- | -------- | ----- |
| S1     | A1       | 7        | 50 分 |
| S2     | A2       | 8        | 40 分 |
| S3     | A3       | 9        | 30 分 |
| S4     | A4       | 10       | 20 分 |
| S5     | A5       | 11       | 10 分 |

### 🔊 音效對應

| 曲目 | 觸發時機     |
| ---- | ------------ |
| 02   | 每次進球     |
| 03   | 超時遊戲結束 |

## 🛠 硬體需求

### 必要元件

- **Arduino UNO** x1
- **TCRT5000 紅外線感測器** x5
- **TM1637 四位數顯示器** x1
- **MP3 音效模組** x1（SoftwareSerial 協定）
- **按鈕開關** x1（保留，備用）

### 線路連接

| 功能         | Arduino 引腳                  |
| ------------ | ----------------------------- |
| 按鈕         | 數位 2 (INT0) + GND，內建上拉 |
| TM1637 CLK   | 數位 3                        |
| TM1637 DIO   | 數位 4                        |
| MP3 模組 TX→ | 數位 5 (SoftwareSerial RX)    |
| MP3 模組 RX← | 數位 6 (SoftwareSerial TX)    |
| IR S1 數位   | 數位 7，內建上拉              |
| IR S2 數位   | 數位 8，內建上拉              |
| IR S3 數位   | 數位 9，內建上拉              |
| IR S4 數位   | 數位 10，內建上拉             |
| IR S5 數位   | 數位 11，內建上拉             |
| IR S1 類比   | A1                            |
| IR S2 類比   | A2                            |
| IR S3 類比   | A3                            |
| IR S4 類比   | A4                            |
| IR S5 類比   | A5                            |

> **紅外線感測器保護**：數位輸入使用 `INPUT_PULLUP`，感測器斷線時引腳維持 HIGH，避免浮空誤判造成錯誤計分。

## 🔧 系統架構

### 狀態機

```
IDLE ──(首球進洞)──→ COUNTING ──(60秒無進球)──→ FINISHED ──(2秒後)──→ IDLE
         ↑                  │ (再次進球，重置視窗)
         └──────────────────┘
```

| 狀態     | TM1637 顯示  | 行為                         |
| -------- | ------------ | ---------------------------- |
| IDLE     | 燈光繞圈特效 | 任一感測器偵測到球自動開始   |
| COUNTING | 當前累積分數 | 進球累加分數並重置 60 秒視窗 |
| FINISHED | 最終分數     | 播放結束音效，2 秒後歸零     |

### 核心技術

- **Timer1 硬體中斷**：1 秒精確觸發，管理 60 秒計時視窗
- **IR 防誤判**：每個感測器獨立 1 秒暫停偵測期 + 200ms 最小間隔
- **INPUT_PULLUP**：感測器斷線保護，防止浮空誤觸發

### 可調整參數

```cpp
const int GAME_TIME = 60;          // 計時視窗（秒）
const int IR_SCORE_VALUES[5] = {50, 40, 30, 20, 10}; // 各位置得分
const unsigned long LIGHT_DELAY = 100; // 燈光特效切換間隔(ms)
```

## 🚨 故障排除

| 問題           | 可能原因             | 處理方式                               |
| -------------- | -------------------- | -------------------------------------- |
| 感測器誤觸進球 | 感測器斷線浮空       | 已由 `INPUT_PULLUP` 保護，檢查實體線路 |
| 計時不準確     | Timer1 中斷被阻塞    | 避免在主迴圈使用長時間 `delay()`       |
| Serial 無輸出  | 波特率不符           | 確認設定為 9600 baud                   |
| 音效不播放     | MP3 模組接線或初始化 | 檢查引腳 5/6 與模組 TX/RX 對應         |

## 🤝 貢獻指南

- 使用兩格縮排、Allman 括號風格
- 常數用 `UPPER_SNAKE_CASE`，函數與全域變數用 `lowerCamelCase`
- ISR 內只設置 `volatile` 旗標，實際處理在主迴圈
- 提交訊息使用繁體中文，說明影響的子系統
